# Vue 3 + TypeScript + Vite

### 1. 패키지 설치 / 삭제
```
yarn install

yarn install -D

yarn remove
```
### 2. 프로젝트 실행
```
yarn dev
```

### 3. 프로젝트 빌드
```
yarn build
```

### 4. 폴더 설명
```
- assets: 정적 리소스 (스타일, 이미지 등)
- components: 재사용 가능한 UI 컴포넌트
- views: 페이지 단위 컴포넌트
- router: 라우팅 설정
- store: 상태 관리
- services: API 통신 로직
- utils: 순수 유틸리티 함수
- types: TypeScript 타입 정의
- constants: 상수 값 정의
- composables: Vue Composition API 관련 로직
```
